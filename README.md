# Codealfa_hangman_game# CodeAlpha_HangmanGame

## 📌 Project Description
This is a simple text-based Hangman Game developed using Python.  
The player has to guess the hidden word one letter at a time before running out of chances.

## 🚀 Features
- Random word selection
- Maximum 6 wrong guesses
- User-friendly console interface
- Tracks guessed letters
- Displays win or lose message

## 🛠 Technologies Used
- Python
- Random module

## ▶️ How to Run
1. Install Python
2. Open terminal or command prompt
3. Run the file using:

python hangman.py

## 📚 Concepts Used
- Loops
- Conditional Statements
- Lists
- Strings
- Random Module
